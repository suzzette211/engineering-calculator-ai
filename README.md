Engineering Calculator API

Description
- A simple API built with FASTAPI to perform basic engineering calculations

Features
. Beam reaction calculator
. Stress calculator
. Unit conversion
. bending moments calculator

Installation
on your terminal
- pip install fastapi uvicorn

RUn the roject
- uvicorn main:app --reload

Access API
. http://127.0.0.1:8000
. http://127.0.0.1:8000/docs
